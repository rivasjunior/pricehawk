# ⚡ PriceHawk

**Monitore preços do Mercado Livre em tempo real — no celular, tablet e PC.**

🔗 **App ao vivo:** https://pricehawkrivas-ot6aaxgy0-rivasjuniors-projects.vercel.app

---

## O que é isso?
Um PWA que roda no celular, tablet e PC.
Você digita o nome de um produto e ele busca os melhores preços no Mercado Livre em tempo real.

---

## PASSO 1 — Criar conta no GitHub (gratuito)

1. Acesse **github.com**
2. Clique em **Sign up**
3. Crie uma conta com seu e-mail
4. Confirme o e-mail

---

## PASSO 2 — Subir os arquivos para o GitHub

1. Faça login no GitHub
2. Clique no **+** no canto superior direito → **New repository**
3. Nome: `pricehawk`
4. Deixe como **Public**
5. Clique em **Create repository**
6. Na próxima tela, clique em **uploading an existing file**
7. Arraste a pasta `pricehawk` inteira (ou os arquivos dentro dela)
8. Clique em **Commit changes**

---

## PASSO 3 — Deploy no Vercel (gratuito)

1. Acesse **vercel.com**
2. Clique em **Sign up** → escolha **Continue with GitHub**
3. Autorize o Vercel a acessar seu GitHub
4. Clique em **Add New Project**
5. Selecione o repositório `pricehawk`
6. Clique em **Deploy**

✅ Em 1-2 minutos seu app estará no ar com uma URL do tipo:
`https://pricehawk-seuusuario.vercel.app`

---

## PASSO 4 — Usar no iPad

1. Abra o **Safari** no iPad
2. Acesse a URL do Vercel (ex: `https://pricehawk-xxx.vercel.app`)
3. Toque em **Compartilhar** (ícone de quadrado com seta) → **Adicionar à Tela de Início**
4. O app vai aparecer como ícone na tela inicial, igual a um app nativo! 📱

---

## Como usar o app

1. **Digite o nome do produto** na barra de busca (ex: "iPhone 15 Pro")
2. Toque em **Buscar** — aparece a lista com preços do Mercado Livre
3. Toque em **+ Monitorar** em qualquer produto
4. Defina um **preço alvo** (opcional) — ex: 4500
5. Na aba **Monitorando**, toque em **⟳ Atualizar Preços** para checar os preços novamente
6. Quando o preço atingir o alvo, aparece 🔔 e um destaque verde

---

## Dúvidas frequentes

**Os preços são reais?**
Sim! Vêm direto da API oficial do Mercado Livre, sempre atualizados.

**Precisa pagar algo?**
Não. GitHub, Vercel e a API do Mercado Livre são 100% gratuitos para este uso.

**Funciona no iPhone também?**
Sim, em qualquer navegador moderno.

**Os produtos monitorados somem?**
Ficam salvos no dispositivo (localStorage). Se trocar de aparelho, precisa adicionar novamente.
